# Cohort Widget

A video recording widget that allows users to watch a video and record their response.

## Files Structure

```
/
├── cohort-index.html
├── cohort-styles.css
├── cohort-script.js
├── vercel.json
└── README.md
```

## Deployment Instructions

1. Make sure all files are in the root directory of your repository
2. Import the repository to Vercel
3. Vercel will automatically detect the configuration from `vercel.json`
4. The site should be deployed and accessible at your Vercel URL

## Local Development

To run locally:
1. Clone the repository
2. Open `cohort-index.html` in a web browser
3. The widget should appear and function as expected

## Dependencies

- No external dependencies required
- Uses native browser APIs for video recording
- Compatible with modern browsers
